# Python Recursion Practice: From Beginner to Moderate

**Rule #1:** You cannot use `for` or `while` loops.\
**Rule #2:** You must have a base case (an `if` statement that stops the
function).


### 1. The Broken Record

Task: Write a function `repeat_msg(message, n)` that prints the string
`message` exactly `n` times.

Input:

    repeat_msg("Hello", 3)

Output:

    Hello
    Hello
    Hello

### 2. The Sum Up

Task: Write a function `sum_n(n)` that returns the sum of all integers
from `1` to `n`.

Input:

    sum_n(5)

Output: 15

### 3. Count the Digits

Task: Write a function `count_digits(n)` that returns how many digits
are in a positive integer `n`.

Input:

    count_digits(5892)

Output: 4

### 4. Recursive Length

Task: Write a function `custom_len(lst)` that returns the length of a
list.

### 5. Find the Number

Task: Write a function `is_present(lst, target)` that returns True if
target is in lst.

### 6. Count Character Frequency

Task: Write a function `count_char(string, char)` that counts char
occurrences.

### 7. Multiply List

Task: Write a function `product_list(numbers)` that multiplies all
numbers.


### 8. Is It Sorted?

Task: Write a function `is_sorted(numbers)` that checks if list is
ascending.

### 9. Fibonacci

Task: Write a function `fib(n)` that returns nth Fibonacci number.

### 10. Sum of Nested Lists

Task: Write a function `deep_sum(lst)` that sums numbers in nested
lists.
